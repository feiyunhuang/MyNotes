## 第三章

* 其他类型和 Boolean 之间的转换

|  数据类型  | 转换为 true 的值 | 转换为  false 的值 |
| :----: | :---------: | :-----------: |
| String |     非空      |      ""       |
| Number |   非零和无穷大    |    0 和 NaN    |
| Object |    任何对象     |     null      |

### 位操作

#### 左移

```javascript
<<

2 << 5 = 64
-2 << 5 = -64
// 左移不会改变符号
```

#### 有符号右移

```javascript
>>
// 符号位不动
```

#### 无符号右移

```javascript
>>>
// 符号位一起移动
```

#### 参数

* 函数定义的参数和传入的参数并不需要一一对应，可以通过 argument[*] 访问。
* func( p0, p1, p2, p3, …… ) 和 argument
